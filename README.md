# Vue interactive with Gin Demo

## 1 Start backend Golang Gin

```sh
go run main.go
```

## 2 Front Web Project Setup

```sh
npm create vue@latest
```

```sh
cd vue_front
npm install
npm install axios
```

```sh
npm install axios
```

### IDE env Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
